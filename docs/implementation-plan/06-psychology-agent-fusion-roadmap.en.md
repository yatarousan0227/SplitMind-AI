# Phase 6: Psychology x AI Agent Fusion Roadmap

## Why This Document Exists

Earlier work on indirect expression improved surface naturalness, but it did not fully address a deeper issue: the system could still feel like an AI filling structured JSON and then composing a response.

## Structural Diagnosis

The roadmap argues that three things were still too weak:

1. psychology existed more as labels than as control logic
2. agent behavior was not yet driven by a genuinely ongoing psychological process
3. memory, relationship, and mood were not strongly enough connected to action selection

## Roadmap Direction

The next architecture should make the psychodynamic model matter in action choice, not just in descriptive trace output.

That means:

- state must influence policy selection
- appraisal must have downstream consequences
- memory must alter later pressure and not only be stored
- the final wording should be a consequence of policy, not a detached writing pass

## Practical Implication

This roadmap is the bridge between "interesting psychology-flavored prompts" and a real agent architecture with persistent internal causality.
