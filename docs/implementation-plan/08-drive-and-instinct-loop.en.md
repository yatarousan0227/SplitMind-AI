# Phase 8: Drive and Instinct Loop

## Goal

Turn desire and instinct from explanatory labels into persistent motivational state that actively drives later turns.

## Original Problem

Earlier versions compressed pressure too heavily into a single dominant desire label and a scalar affective pressure value. That made it hard for desire to continue shaping action over time.

## Target Change

1. drives persist across turns
2. frustration and inhibition affect later action selection
3. responses feel like they come from wanting something, not only from describing emotion

## Current Snapshot

As of 2026-03-17, the major Phase 8 track is already implemented across:

- evaluation and datasets
- state and contracts
- motivational routing
- memory persistence
- safety boundaries
- the Streamlit dashboard

## Why This Phase Matters

This is the shift from static emotional labeling to ongoing motivational dynamics. It is one of the most important steps in making the system feel less like a classifier and more like an agent with persistent internal pressure.
