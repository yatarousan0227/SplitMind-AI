# Support

## Getting Help

Use the repository issue tracker for:

- bug reports
- feature requests
- documentation gaps
- reproducible setup problems

When asking for help, include:

- your Python version
- provider configuration shape, without secrets
- the command you ran
- relevant logs or trace excerpts
- whether the issue reproduces with `uv run pytest tests/unit -q`

## What This Project Is

SplitMind-AI is a research-oriented framework and reference implementation. It is not positioned as a production SLA-backed service.

## What Maintainers Can Best Help With

- architecture questions grounded in the repository
- bugs with clear reproduction steps
- evaluation and trace interpretation
- memory, state, and safety behavior

## What Maintainers May Not Be Able To Support

- custom hosted deployments
- provider billing or account issues
- large private integration work
